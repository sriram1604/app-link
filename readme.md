## 🏢 Features

### Attendance Management
- **Biometric Login**: Secure authentication using fingerprints or face recognition for error-free attendance.
- **Geo-Fencing**: Track attendance only within specified office zones using high-precision GPS.
- **Smart Notifications**: Automated reminders for check-ins, check-outs, and pending approvals.

### Workforce Analytics
- **Real-time Analytics**: Get instant insights into workforce presence, overtime, and leave management.
- **Workforce Dashboard**: Track efficiency, attendance, and performance metrics.

### Workforce Solutions
- Provide contract-based skilled and unskilled manpower for cardboard manufacturing, packaging units, and industrial operations.
- Ensure output targets are met without disruptions.
